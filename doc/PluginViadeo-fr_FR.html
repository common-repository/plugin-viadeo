Aide Plugin Viadéo
  • Supprimer un ou plusieur profils

    • Sélectionnez le ou les profils à supprimer, puis sélectionnez l'élément " Supprimer " dans la liste de choix situé en dessous du la liste des profils. Et cliquez sur " Appliquer ".

  • Ajouter un profil

    • Saisissez les champs obligatoires, et cliquez sur " Ajouter ". Attention l'url dont être complète (ex: http://www.viadeo.com/fr/profile/prenom.nom)

  • Modifier un profil

    • Sélectionnez un profil dans la liste, les informations sont automatiquement renseignées dans les champs. Puis, cliquez sur " Modifier ". Attention si vous sélectionnez plusieurs profils les champs sont ré-initialisés automatiquement, passage en mode suppression ou ajout. Vous devez sélectionner un seul profil pour passer en mode modification.

  • Affichage des profils

    • La page d'affichage des profils est automatiquement créée à l'activation du plugin. Le fichier ' Viadeo.css ' permet la personnalisation d'affichage du profil.

  • Suggestions et améliorations

    • N'hésitez pas à laisser vos suggestions ou remarques concernant ce plugin sur le site www.dimgoto.com, afin d'améliorer le plugin.
